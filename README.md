# bfc v 1.1.0

An optimising bf compiler written in Common Lisp.

## Requirements:

 *  SBCL (Makefile can be modified to work with CLISP)
 *  NASM
 *  ld

## Usage Example

```bash
echo "++++++++[>++++[>++>+++>+++>+<<<<-]>+>+>->>+[<]<-]>>.>---.+++++++..+++.>>.<-.<.+++.------.--------.>>+.>++." | make && ./bf
```
